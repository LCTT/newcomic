My (Dev) Morning Routine
===============

![My (Dev) Morning Routine](http://turnoff.us/image/en/my-morning-routine.png)

via:

- My (Dev) Morning Routine:
- Open Eclipse
- Change The World
- Eclipse Not Ready
- Change The World again...Turn back to  the original state
- Eclipse almost ready
- Little Nap
- Eclipse up
- Start Coding
- Eclipse Encourages Meditation
