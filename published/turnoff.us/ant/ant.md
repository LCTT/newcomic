蚁族：我们用“人”来构建软件
===============

![Coding From Anthill](https://github.com/LCTT/comic/raw/master/turnoff.us/ant/ant.png)

- 某时，在某个蚁丘里面……
- 嗨，哥们，你在用什么做构建？
- 人啊。

------

用过 Java 的同学大多都用过或听说过 Ant，它是用来构建 Java 项目的一个工具。

那么，在蚁族的世界中，他们（它们？）是不是用“Human（人）”来构建项目呢？
